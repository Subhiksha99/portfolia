Social Distance Monitoring

From CCTV video , we can monitor whether people are following social distance or not.Using OpenCV library , frames are extracted and objects are detected using YOLO.Then human beings are figured out and distance between them is calculated.Django is used to develop a website to monitor people whether everyone follows social distance or not.Alert messages are given to the corresponding person when there is more violation.Twilo is used for alert messages.

Handwritten Recognition using Neural Network

The model is trained using IAM dataset to detect handwritten text.The recognised text is converted to other language using google translator.The model is trained with some of my own handritten words and able to detect the words correctly.OpenCV contour concept is used to detect word from sentence and given this as an input to the model.

Milk Quality Detection

The novel algorithm was introduced to reduce the number of nodes in hidden layer of ELM using Logistic Regression.The Quality of milk is detected using FTIR which gives the spectral point.The spectral points are given as an input to ELM network and determine the adulterant present in the milk.Flask is used to create a website to check the milk Quality.
