Online Shopping Site

Online electronics shopping website is created using JAVA, JSP, SERVLET, JDBC, JQUERY, HTML and CSS. In this website user can login to view the items and buy the electronics items of their wish.User can search the particular product and can view that item. They can send the queries to admin. Customer queries and complaints are stored in db and can view by admin. UI was developed with the help of bootstrap and CSS .Cookies were used to store user name and used to greet users. Users details were stored in MYSQL DB to track user shopping list. Electronics items were stored in db with image and more technical information about the products. The user can view all the products and they can select one product and view it in separate page with more details.
